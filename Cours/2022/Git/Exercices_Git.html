Exercices - Git

Pour ces exercices, en plus de taper dans le terminal vos commandes, veuillez à me rendre cet exercice résolu sous forme numérique: bloc note, document word, etc.

Exercice n°1

Partie 1 - Création d’un dépôt

  1. Créez un répertoire nommé premierDepot
  2. Fais-en un dépôt.

Partie 2 - Ajouter de fichiers

  1. Copiez 3 fichiers de votre choix: html, js, txt, etc.

Partie 3 - Indexer les fichiers

  1. Indexez le premier fichier avec git.
  2. Indexez les autres fichiers avec une seule commande git.

Partie 4 - Validation/Commit

Validez les fichiers présents dans la zone d’index avec le message suivant:“Commit initial” (C’est souvent le premier message que l’on donne quand pour le tout premier commit)

Partie 5 - Modification

Modifiez un des trois fichiers avec une modification qui ait du sens.

Partie 6 - Indexation

Indexez le fichier.
